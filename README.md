# Ansible Devops

## DevOps - Demo Builder

Repository containing self-contained playbooks to install the following applications:

- Ansible Tower
- AWX
- FreeIPA
- GitLab
- Jenkins
- Artifactory
- Darkbulb-NXOS
- Darkbulb-IOS
- Darkbulb-EOS
- Darkbulb-JUNOS
- oVirt (on-going)
- Openshift (soon)

Cloud Providers:

- Google Compute Platform
